# 💼 Job Board Platform (CodeAlpha Internship Project)

A backend Job Board system built using Node.js, Express, and MongoDB.  
It allows users to post jobs, view job listings, and apply for jobs using REST APIs.

---

## 🚀 Features

### 🧑‍💼 Jobs
- Create job postings
- View all jobs
- View single job details

### 📝 Applications
- Apply for a job
- View all applications
- Filter applications by job

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- dotenv
